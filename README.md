# ほしぞらさんぽ 🌌

こどもとおとなのための Web プラネタリウム。実際の日時・場所の星空を天文計算で再現し、星座をタップすると由来・神話が読めます。

**▶ https://youheioonuki.github.io/hoshizora-sanpo/**

## 特徴

- **リアルタイム星空** — 主要恒星約140星（J2000 赤経・赤緯・等級・B-V色指数）を GMST → 地平座標変換 → ステレオ投影で描画。太陽位置も計算し、昼夜・朝焼け夕焼けを再現
- **27星座の由来・神話** — 黄道12星座＋オリオン・北斗七星・南十字星などの有名星座。タップで解説シート
- **ひらがなモード** — 5歳から読める分かち書きひらがな解説。星座名・方角・ボタンもすべてかな表示
- **時間たび** — スライダー／±1日／±1ヶ月／600倍速の早送りで日周運動・年周運動を体感
- **場所切り替え** — 東京・札幌・那覇・シドニー。南半球では星が逆回りに見え、南十字星が現れる
- **星座クイズ** — 「〇〇座をさがしてタップ！」形式の5問ゲーム
- 演出: 星の瞬き・天の川・流れ星・アンドロメダ銀河・すばる・夏冬の大三角ガイド線

## 使い方

スマホ推奨。ドラッグで見まわし、ピンチでズーム、星座をタップで解説。
ブラウザの「ホーム画面に追加」でアプリのように起動できます（PWA マニフェスト同梱）。

## 技術

- 依存ライブラリゼロの単一 HTML + Canvas 2D（`index.html` がすべて）
- ビルド不要。静的ホスティングのみで動作

## 注意

教育用途のプロトタイプです。暗い星の一部座標は近似値を含みます。星座の神話はギリシャ神話などの一般的な伝承に基づく要約です。
